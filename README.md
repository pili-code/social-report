# The Design Project - Social Media Analytics

Live dashboard data for TDP's social media performance tracking.

## Structure

```
├── data/                    # Raw CSV data (append-only, grows over time)
│   ├── weekly_summary.csv   # Cross-platform weekly totals
│   ├── youtube_videos.csv   # Video-level YouTube data
│   ├── tiktok_daily.csv     # Daily TikTok metrics
│   ├── x_daily.csv          # Daily X/Twitter metrics
│   ├── meta_campaigns.csv   # Meta Ads campaign data
│   ├── linkedin_posts.csv   # LinkedIn post metrics
│   ├── linkedin_demographics.csv  # LinkedIn audience insights
│   └── customer_wins.csv    # Client wins from social
├── reports/
│   ├── latest.json          # Current report (V0 dashboard reads this)
│   ├── latest.xlsx          # Downloadable Excel report
│   └── archive/             # Historical reports
└── dashboard/
    └── config.json          # Dashboard configuration
```

## Dashboard Feed

The V0 dashboard reads `reports/latest.json` which contains:
- Cross-platform performance comparison
- Week-over-week changes
- Platform-specific detail (YouTube videos, TikTok daily, etc.)
- LinkedIn demographics for ICP validation
- Customer wins tracking
- Historical trends for charts
